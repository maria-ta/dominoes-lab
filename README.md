# Software engineering methodologies

## Description

Homework for **Software engineering methodologies** course.

Node.js and npm are required to run the project. [Install Node.js and npm](https://nodejs.org/en/).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

After application start, open http://localhost:3000/index.html

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Lint

```bash
$ npm run lint
```
